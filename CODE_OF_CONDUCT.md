# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version: be kind, be patient, assume good faith, critique code not people, and remember there's a human on the other side of every issue and PR. The usher seats everyone.

Unacceptable behavior (harassment, personal attacks, trolling, doxxing) can be reported privately via [GitHub's report abuse tools](https://github.com/contact/report-abuse) or a [private security advisory](https://github.com/theodorebeaupre-prog/usher/security/advisories/new) marked "conduct"; the maintainer will respond within a few days. Consequences scale from a warning to a permanent ban at the maintainer's discretion, per the Covenant's enforcement guidelines.
